# AI-Based Student Placement Prediction System

## Overview
This project is a machine learning-based web application developed using Python and Flask.

The system predicts whether a student is likely to get placed based on:
- CGPA
- Internships
- Projects
- Communication Skills
- Aptitude Score

The application also provides personalized suggestions for improvement.

- Python
- Flask

- CSS

- Scikit-learn
